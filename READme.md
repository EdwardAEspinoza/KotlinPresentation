**How to Run in GitHub Codespaces**

Open your Codespace in the repository.

- Go to the src folder:

- cd src


**Make sure Kotlin is installed (if not, run this once):**

- `sudo apt update`
- `sudo apt install kotlin -y`


**Compile the program:**

- `kotlinc AdventureGame.kt -include-runtime -d adventuregame.jar`

- Run the Program:

  - `java -jar adventuregame.jar`

- `kotlinc BankingSystem.kt -include-runtime -d bankingsystem.jar`

- Run the Program:
  -  `java -jar bankingsystem.jar`
